# 📊 TradeZone Business Performance Analysis (2023–2024)

## Overview
This project analyzes business performance using SQL to uncover insights across customer behavior, product performance, seller efficiency, and revenue trends.

## Tools Used
- SQL (PostgreSQL)
- Excel

## Key Insights
- Revenue growth driven by increased order volume
- High spenders contribute the majority of revenue
- Regional differences affect conversion rates

## Files Included
- Data Cleaning Script
- SQL Queries (Q1–Q8)
- Analyst Memo
